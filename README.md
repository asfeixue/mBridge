# mBridge
衔接协议双方，并提供可靠支持

## 0.1.0-SNAPSHOT
### 支持协议友好展示
        1.      支持基于系统来索引协议
        2.      支持订阅header，实时watch住最新的header
        3.      支持针对单点数据item做一定的表达式校验
                
### 支持客户端mock联调
        1.      支持查看客户端请求header&body
        2.      支持返回指定的mock响应
        3.      支持针对请求数据的结构校验
        
### 支持服务端mock测试
        1.      支持针对服务端协议的mock测试
        2.      支持针对服务端协议测试响应的结构校验
